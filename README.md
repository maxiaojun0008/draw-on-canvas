# draw-on-canvas

A TypeScript monorepo for building an open-source canvas image annotation library.

## Workspace packages

- `@draw-on-canvas/core`: shared types and core logic
- `@draw-on-canvas/canvas2d`: Canvas2D renderer package
- `@draw-on-canvas/demo`: local demo app powered by Vite

## Development

```bash
pnpm install
pnpm lint
pnpm test
pnpm build
```
